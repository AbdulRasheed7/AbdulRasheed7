## 🚀 Featured Projects

### 🏦 [Institutional Financial Analytics Dashboard — State Bank of Pakistan](https://github.com/AbdulRasheed7)
> **Python · Streamlit · Oracle XE · Plotly · Pandas**

Built a production-grade, multi-page **Streamlit dashboard** for SBP's Statistics & Data Services Department. Covers 5 years (2019–2023) of financial data across 8+ institution types (Banks, DFIs, MFBs, Insurance, Modarabas, etc.). Features KPI tiles (ROA, ROE, Equity/Asset ratio), waterfall profit/loss charts, multi-year trend lines, and Excel export — branded to SBP's visual identity. Connected to a live **Oracle XE** database with dynamic schema detection.

---

### ✈️ [Customer Loyalty & Flight Activity BI Dashboard — FlyingWhale Airline](https://github.com/AbdulRasheed7)
> **Power BI · DAX · Data Modelling**

Multi-page **Power BI** report analyzing **2M+ total flights** and **14,670+ active loyalty members** across CLV tiers, enrollment patterns, and cancellation trends. Engineered DAX-calculated columns for Enrollment Duration, implemented drill-down filters, narrative visuals, and map-based cancellation analysis by province.

---

### 🏃 [Treadmill Buyer Profile Analysis — AeroFit EDA](https://github.com/AbdulRasheed7)
> **Python · Pandas · Matplotlib · Seaborn**

End-to-end **Exploratory Data Analysis** on 180 customer records. Identified key correlations (Income vs. Miles: r = 0.58, Fitness vs. Usage: r = 0.79), computed conditional probabilities (78% of KP781 buyers are high-income), and delivered actionable segmentation and marketing recommendations.

### 🔍 [Murder Mystery SQL Investigation — AtomCamp](https://github.com/AbdulRasheed7)
